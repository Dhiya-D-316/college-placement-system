# College Placement Management System

## Tech Stack
- MySQL
- phpMyAdmin
- XAMPP

## Features
- Student & Company Management
- Placement Tracking
- ACID Transactions (Atomicity, Consistency, Isolation, Durability)
- Foreign Keys & JOIN Queries

## Tables
- **students** - Student_id, Name, CGPA, Dept
- **companies** - Company_id, Name, CTC
- **placements** - Placement_id, Student_id, Company_id, Date

## Queries Used
- CRUD Operations (CREATE, READ, UPDATE, DELETE)
- WHERE Conditions
- JOIN Queries
- Transaction Management (COMMIT & ROLLBACK)
